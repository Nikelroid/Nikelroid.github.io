# kelidari.com

Personal site of Nima Kelidari — machine learning research & engineering.

Plain HTML and CSS, no framework, no build step. Fonts are self-hosted
(Fraunces, Newsreader, IBM Plex Mono); images are pre-optimized WebP.
Deployed to GitHub Pages by the workflow in `.github/workflows/deploy.yml`,
which publishes the repository contents as-is.

To edit: change the HTML/CSS directly and push to `main`.
